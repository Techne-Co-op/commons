## Question

Whether a publicly visible app shell is acceptable for the alpha.

## Context and options

Proposed default: accept the public static shell; privacy enforced at the data layer; noindex; invite-only auth. Alternative augmentation: a proxy with access control in front of Pages.

## Proposed default

Stated in the context above; a proposal only. The decision is reserved to the named owner.
