## Question

Whether the repository is public from day one.

## Context and options

Proposed default: public from first commit, the O-pathway Phase 01 commitment made literal (CIS §04). Hard rule either way: no secrets, no member PII, synthetic seeds only.

## Proposed default

Stated in the context above; a proposal only. The decision is reserved to the named owner.

## Record of decision

Decided 2026-06-12 by Todd Youngblood (Ventures & Operations Steward).

The repository is public from the first commit. This is the O-pathway Phase 01 commitment made literal: the CIS is built in the open from day one, and the public record is the accountability mechanism. The hard rule holds regardless of visibility — no secrets, no member PII, synthetic seeds only. The repo was already public at github.com/Techne-Co-op/commons before this record was written; this decision formalizes what the build already assumed.
