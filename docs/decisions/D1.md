## Question

Which tool builds the member portal.

## Context and options

PRD criteria: portability, accessibility, mobile-first, Supabase compatibility, design-system fidelity, plus the chokepoint test (CIS §17). Proposed default: a static-first framework with islands of interactivity (Astro is the type specimen; Vite + React SPA is the alternative).

## Proposed default
