## Question

Which service delivers official email.

## Context and options

Required because notices carry legally timed windows and delivery tracking (CIS §08, draft-bylaws §2.4); Supabase built-in mail covers auth flows only. Candidate class: Resend, Postmark, SES; no endorsement implied.

## Proposed default

Stated in the context above; a proposal only. The decision is reserved to the named owner.

## Record of decision

Decided 2026-06-12 by Todd Youngblood (Ventures & Operations Steward).

Resend, as the default for alpha. Free tier (3,000 emails/month) covers the cooperative's alpha volume; API is straightforward to integrate with Supabase Edge Functions; delivery webhooks satisfy the tracking requirement for legally timed notices. Revisit if volume, compliance requirements, or deliverability issues warrant Postmark. SES is not adopted — operational overhead contradicts D5. Supabase built-in mail remains in use for auth flows only.
